# WiSH — Texas Instruments Mentorship Program

Welcome to the WiSH (Women in STEM and Hardware) mentorship repository for Texas Instruments.

## Repository Structure

```
WiSH-Texas-Instruments/
└── WiSH_2026/                  # All student work for the 2026 cohort
    ├── introductions/           # One .txt file per mentee — your intro goes here
    ├── calculator_project/      # Week 2 team exercise (C calculator)
    └── ...                      # Files you create during the exercises
```

## Getting Started

1. **Fork** this repository to your own GitHub account.
2. **Clone** your fork locally:
   ```bash
   git clone git@github.com:<your-username>/WiSH-Texas-Instruments.git
   cd WiSH-Texas-Instruments
   ```
3. **Navigate** into the 2026 activities directory — this is where all your work goes:
   ```bash
   cd WiSH_2026
   ```
4. Create a branch for your work:
   ```bash
   git switch -c <your-name>/wish-2026
   ```

## Contributing

- All student work is submitted as a Pull Request from your fork to `Raaghav-Arya/WiSH-Texas-Instruments` main.
- Name your PR: `Add <your-name> introduction — WiSH 2026`
- One file per person in `introductions/`; one team branch per team for the calculator project.
